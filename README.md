TINY TASTES — Persian Restaurant website
=========================================

A single-page website for Tiny Tastes (Perzská reštaurácia), Nivy Tower, Bratislava.
Classic Persian styling — teal, gold and parchment — matching your menu artwork.

FILES
-----
  index.html        The website (open this in any browser to preview).
  css/styles.css    All styling — colours, fonts, layout.
  js/main.js        Mobile menu, gentle scroll animations.
  assets/           Favicon (browser-tab icon).

HOW TO PREVIEW
--------------
  Just double-click index.html — it opens in your browser. No setup needed.

HOW TO PUT IT ONLINE
--------------------
  Upload the whole folder (keeping the css/js/assets structure) to any host,
  e.g. Netlify (drag-and-drop the folder), Vercel, GitHub Pages, or your
  domain provider's file manager. index.html must stay at the top level.

THINGS YOU'LL LIKELY WANT TO EDIT (all in index.html)
-----------------------------------------------------
  - Opening hours (Visit section) — set your real times.
  - Order links — the Foodora / Wolt buttons currently point to the
    homepages; replace with your restaurant's direct order links.
  - Address / phone — confirm the exact unit and add a phone number if wanted.
  - Instagram link is already set to @tinytastes_sk.
  - Dish descriptions — I wrote short English lines under each Slovak name;
    adjust to match your kitchen.

Fonts load from Google Fonts, so an internet connection shows them best.

Made with Persian hospitality.
